# 基于 RDK X5 与多模态光电感知的双机协同智慧康养机器人

## 功能介绍

面向独居老人居家养老场景的异构双机智慧康养系统：六足机器人负责巡检建图，升降机械臂大车承担边缘计算与物品抓取，自研"颐伴"多模态光电感知板完成环境安全监护，决策层运行于地平线 RDK X5 边缘计算平台，各节点经 ROS 2 协同通信。

核心功能：

1. **多模态光电安全监护** —— 光照、温湿度（DHT11）、火焰、烟雾（MQ-2）四类信号采集，DO 速判 + AO 精判两级判定，越限直驱蜂鸣器本地告警（不依赖网络）
2. **AI Agent 康养管家** —— RDK X5 上运行基于开源 OpenClaw 框架二次开发的康养 Agent，提供用药提醒、识物问答与日常陪伴
3. **异构双机协同** —— 六足巡检建图 + 大车目标趋近与抓取，经 ROS 2 DDS 分布式部署
4. **低门槛跨端交互** —— 自然语言经 MCP 工具调用转化为 NEC 红外家电控制与 ADB 微信视频呼叫/投屏

本仓库收录系统的嵌入式侧源码（MCU 固件与六足平台 ROS 2 工作空间）。

## 支持平台

| 平台 | 角色 | 开发环境 |
|---|---|---|
| 地平线 RDK X5 | 决策层边缘计算（AI Agent、视觉理解） | Ubuntu / ROS 2 |
| 树莓派 4B | 六足平台上位机（SLAM 建图、导航） | Ubuntu / ROS 2 |
| STM32F103C8T6 | 光电感知板、升降平台控制 | Keil MDK5 |
| STM32F10x / F407VET6 | 机械臂舵机控制、大车底盘 | Keil MDK5 |

## 物料清单

- 六足机器人平台（18 路总线舵机、激光雷达、深度相机、IMU、麦克风阵列）
- 机械臂大车平台（轮式底盘、丝杠升降 Z 轴、6 自由度机械臂、RDK X5）
- 自研"颐伴"光电感知板（STM32F103C8T6 + DHT11 / 红外火焰 / MQ-2 / 光敏，AO+DO 双通道插座化接口）
- 无线串口模块、锂电池组

## 使用方法

### MCU 固件（sensor_v2.1 / PWM机械臂 / 伺服电机控制 / 大车代码）

1. 安装 Keil MDK5 及对应器件包（STM32F1xx / STM32F4xx DFP）
2. 打开各目录下的 `.uvprojx` / `.uvproj` 工程编译
3. 经 ST-Link / J-Link 烧录

升降平台的串口 ASCII 指令协议详见 [伺服电机控制/readme.md](伺服电机控制/readme.md)。

### 六足平台 ROS 2 工作空间

```bash
cd 六足机器人代码
colcon build
source install/setup.bash
# 依次启动雷达、IMU、底盘桥接与建图/导航 launch
```

## 原理简介

系统采用"感知—决策—执行—交互"四层闭环架构：感知层由激光雷达、深度相机、麦克风阵列与自研光电感知板构成；决策层以 RDK X5 为核心运行 AI Agent 并接入大语言模型；执行层由 STM32/GD32 完成底盘、机械臂与升降平台的实时控制；交互层实现视频呼叫、家电联动与语音播报。各层经 ROS 2 话题/服务解耦，告警类话题采用 RELIABLE、高频传感话题采用 BEST_EFFORT 的 QoS 策略。
